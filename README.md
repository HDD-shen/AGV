# AVG 目标路径算法
基于python的AGV路径规划算法
## 项目简介
这是一个高效的 多智能体路径规划（MAPF）系统 ，采用MAPF-PUSH算法，能够在复杂地形环境中为多个AGV（自动导引车）规划无冲突的最优路径。

## 核心算法
### MAPF-PUSH 算法
- 运行速度快 ：比传统CBS算法快5-10倍
- 成功率高 ：在复杂环境中表现优异
- 内存占用低 ：无需维护大规模冲突树
### 寻路算法选择
1. A * - 标准A*算法，稳定可靠
2. A v2 * - 改进版，优先选择最优邻居
3. Weighted A * - 加权版本，支持地形代价计算
## 主要特性
- ✅ 复杂地形支持 ：草地、沙地、水域、道路等多种地形
- ✅ 对角线移动 ：支持8方向移动
- ✅ 实时可视化 ：Pygame图形界面演示
- ✅ 交互控制 ：播放/暂停/单步/调速
